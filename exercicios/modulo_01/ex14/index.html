CSS: estilos locais / internos

No CSS com estilos locais / internos todos os estilos são atribuídos dentro da tag style que é colocada no head do código HTML. Depois é só criar uma área específica (por exemplo: body, h1, h2, p, ...) e definir os estilos para cada uma delas.
