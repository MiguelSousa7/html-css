Principais formatações

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag <b>. Porém o negrito não é semântico, é apenas uma forma.

Nesta frase temos um termo em destaque usando a tag <strong>. Embora visualmente seja igual ao negrito, a tag strong é semântica.

Itálico / Ênfase

Nesta frase temos um termo em itálico usando a tag <i> (não semântica)

Nesta frase temos um termo em ênfase usando a tag <em> (semântica)

Texto marcado

Podemos também criar um texto marcado usando a tag <mark>.

Neste parágrafo mudamos a cor do marcador. Mas isto já é CSS.

Texto grande e pequeno

Estamos a criar um texto grande e → um texto pequeno ← neste parágrafo.

Texto apagado

Podemos marcar um texto como excluído, para indicar q deve ser lido mas não considerado, com a tag <del>

Texto inserido

Podemos marcar um texto como inserido, para dar destaque e indicar que ele foi adicionado posteriormente com a tag <ins>(semântico).

Também existe texto sublinhado com a tag <u>, porém já está obsulento(não semântico).

Texto subrescrito

Para inserir coisas do tipo numeors com expoente: x20+3

Texto subscrito
