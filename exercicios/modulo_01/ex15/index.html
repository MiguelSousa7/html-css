CSS: estilos externos

No CSS com estilos externos coloca-se todo o código relacionado a CSS numa página chamada style.css. Depois cria-se no head, de todas as páginas q usufruem desse código CSS, um "link:css" que fará ligação com a página q contém o código CSS. Desta forma, basta mudar o código da página q contém o CSS para q o estilo mude em todas as páginas, ao invés de ter que mudar individualmente.
